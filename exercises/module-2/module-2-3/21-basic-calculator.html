Basic Calculator

Description

Build a simple calculator program that takes two numbers from the user and performs all four basic mathematical operations (addition, subtraction, multiplication, and division). Your program should display the results in a clean, formatted way.

Tasks

Your program must:

  1. Welcome the user to the calculator
  2. Ask the user to enter two numbers (support both integers and decimals)
  3. Perform all four basic operations: +, -, *, /
  4. Display the results using formatted strings
  5. Handle division by zero appropriately
  6. Round division results to 2 decimal places for cleaner output

Expected Output Format

            Welcome to the Simple Calculator!
            Enter the first number: 5
            Enter the second number: 3
            5.0 + 3.0 = 8.0
            5.0 - 3.0 = 2.0
            5.0 * 3.0 = 15.0
            5.0 / 3.0 = 1.67

Step-by-Step Guide

Step 1: Set up user input

  • Print a welcome message
  • Use input() to get two numbers from the user
  • Convert the input strings to numbers (hint: consider using float() to handle decimals)

Step 2: Perform calculations

  • Create variables to store the results of each operation
  • Perform addition, subtraction, multiplication, and division

Step 3: Display results

  • Use f-strings to format your output nicely
  • Show each operation and its result

Step 4: Handle edge cases

  • Add a check to prevent division by zero
  • Use round() to make division results cleaner (2 decimal places)
