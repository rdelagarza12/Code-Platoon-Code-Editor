Exercise 5: Variable Reassignment

Create a program that swaps the values of two variables. The program should ask the user for two values, store them in variables a and b, swap their values, and then display the values before and after swapping.

Example Input

    First value (a): 10
    Second value (b): 20

Example Output

    Before swapping: a = 10, b = 20
    After swapping: a = 20, b = 10

Tasks

  1. Create variable a and b to store the user's input values
  2. Display the values before swapping
  3. Swap the values of a and b using a temporary variable
  4. Display the values after swapping
