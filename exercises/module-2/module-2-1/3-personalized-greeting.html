Exercise 3: Personalized Greeting

Write a program that asks the user for their name and then prints a personalized greeting. Your program should:

Tasks

  1. Ask the user to input their name
  2. Store the input in a variable
  3. Create a variable that holds a greeting
  4. Create a seperate variable that joins the greeting and the name
  5. Print a new formatted greeting variable

Example Output:

If the user enters "Alice", your program should output: "Hello, Alice!"
