Exercise 4: Greeting Age

Create a program that asks for the user's name and age, then stores these values in variables. The program should then create a greeting message using these variables and print it out.

Example Input:

        Name: Alice
        Age: 25

Example Output

        Hello Alice! You are 25 years old.

Tasks

  1. Create a variable to store the user's name from output
  2. Create a variable to store the user's age from input
  3. Create a greeting message using these variables
  4. Print the greeting message
