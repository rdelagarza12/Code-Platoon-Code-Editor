Exercise 6: Basic Math

Create a program that takes your grade from three different classes and returns your overall average. The averages is calculated by adding everything together and dividing the sum by the amount of values that were added.

Example Input:

        english = 95
        math = 88
        history = 76

Example Output:

        86.33

Tasks

  1. Create three different variables for your three classes
  2. Create a seperate variable for the sum of all the classes
  3. Create a final variable that calculates the average
  4. Output the result
