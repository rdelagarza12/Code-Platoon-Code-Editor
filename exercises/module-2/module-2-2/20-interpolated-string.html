Exercise 20: String Interpolation

Create a program that creates variables for a Student's school record and use the f-string method to create an Interpolated String.

Notice: When using the f-string you do not need to convert your variables to strings beforehand

Expected Input:

        Students name: "Jason Borne"
        Student ID: "JB6748"
        Student GPA: 3.8

Expected Output:

        Interpolated String: "Name: Jason Borne, ID: JB6748, GPA: 3.8"

Tasks

  1. Create variables for the Student name, id, and gpa
  2. Create a seperate variable using the f-string to create an Interpolated String
  3. Display the students record
