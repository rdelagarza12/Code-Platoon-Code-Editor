Exercise 16: Exponents

Create a program that helps calculate the power of a number by creating a base and exponent

Expected Input:

        base = 2
        exponent = 3

Expected Output:

        2^3 = 8

Tasks:

  1. Create a variable for the base
  2. Create a variable for the exponent
  3. Use the exponent operator ** to calculate the result
  4. Display the result
