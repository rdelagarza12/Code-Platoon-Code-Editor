Exercise 15: Floor Division

You have a group of students and you want to divide them into teams of equal size. Create a program that calculates how many full teams you can form. The floor division does normal division put rounds down to the nearest whole number

Expected Input:

        Number of students: 23
        Number of students per team: 4

Expected Output:

        How many teams: 5

Tasks

  1. Create a variable for number of students
  2. Create a variable for numbers of students per teams
  3. Use floor division // to calculate how many full teams can be formed
  4. Display the result
