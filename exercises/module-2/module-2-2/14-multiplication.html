Exercise 14: Multiplication

Create a program that has two variables, number of pencils, and number of boxes and then calculate the total number of pencils if each box holds the same number of pencils.

Expected Input:

    Number of pencils: 20
    Number of boxes: 4

Expected Output:

    80

Tasks

  1. Create a variable with the quantity of pencils per box
  2. Create a variable with the quantity of boxes
  3. Multiply the two numbers to get the total number of pencils
  4. Display the result
